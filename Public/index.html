Info

Birdwatch is a multiplayer text adventure style game about good and evil, birds, and consequences. To play, sign in and, type commands in the text input on the bottom right side of the screen. If you're ever unsure what to do, type "help" for a list of applicable commands, or ask in chat.

Also try BW Idle Hands!


Changelog

First public version of Birdwatch!


v1.0.0 - Feb 17, 2023

Idle Hands is mostly complete, and added a link to the Birdwatch homepage. The game has a homepage now, too, with supporters, info, and the changelog, which you're reading.


v1.1.0 - Mar 14, 2023

Added a leveling system which caps speed and strength stats (seperately from item buffs) to balance the game a bit better. Also increased the max mining speed buff from 150 to 250.


v1.1.1 - Mar 20, 2023

Added fighting as a way to get XP, and added an XP counter to the "stats" command.
